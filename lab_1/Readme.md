# SSE

## Необходимо:
- При помощи SSE инструкций написать программу (или функцию), которая перемножает массив из 4х чисел размером 32 бита.
- Написать аналогичную программу (или функцию) которая решает ту же задачу последовательно.
- Сравнить производительность
- Проанализировать сгенерированный ассемблер: gcc -S sse.c
