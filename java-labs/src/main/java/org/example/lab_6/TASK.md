# OpenMP (сравнение производительности с Pthreads, последовательным кодом)

## Необходимо создать клиент-серверное приложение:
- Несколько клиентов, каждый клиент - отдельный процесс
- Серверное приложение - отдельный процесс
- Клиенты и сервер общаются с использоваени Socket

## Необходимо релизовать функционал:
- Клиент подключается к серверу
- Сервер запоминает каждого клиента в java.util.concurrent.CopyOnWriteArrayList
- Сервер читает ввод из консоли и отправляет сообщение всем подключенным клиентам
